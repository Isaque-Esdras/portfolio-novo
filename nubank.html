Site desenvolvido com:
  • Java Script | HTML5 | CSS3

    • Fiz esse projeto visando trabalhadores do Call Center, que as vezes precisa cadastrar a quantidade de clientes para quem vendeu durante o expediente. Supervisores ou Gestores de Operação as vezes precisam cadastrar as novas equipes rotativas mensais, essa seria uma interface com funções mais intuitivas para a correria do dia-a-dia do Call Center.
