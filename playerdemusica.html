Site desenvolvido com:
  • Java Script | HTML5 | CSS3

    • Esse foi um projeto player de música, onde desenvolvi a parte visual para acesso do usuário ao player musical. Fica completamente acessível em computador ou mobile. Tentei combinar algumas cores como azul e lilás, as musicas são selecionaveis através do aparelho, podendo ser possível pausar, dar play, regredir e avançar para uma próxima música.
